# Intro to AI: Teaching Resources

This repository contains materials associated with the paper Unplugged AI Activities for Embodied Interactions in Higher Education, published at EAAI-26: The 16th Symposium on Educational Advances in Artificial Intelligence. The paper introduces a set of unplugged learning activities designed to help undergraduate students intuitively understand core AI concepts before transitioning to formal mathematics and programming.
